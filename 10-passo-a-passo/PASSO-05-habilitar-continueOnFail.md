# PASSO 5 — Habilitar continueOnFail nos HTTP de ação

Para cada node abaixo no **Fix Conflito v2**:
1. Clicar no node
2. Settings (engrenagem)
3. Ativar **"Continue On Fail"**
4. Salvar

- HTTP - Create Tool
- HTTP - Create Tool1
- HTTP - Create Tool2
- HTTP - Create Calendar Tool
- HTTP - Create Calendar Tool2
- HTTP - Create Calendar Tool3
- HTTP - Create Calendar Tool4
- HTTP - Create Calendar Tool5
- HTTP - Create Calendar Tool6
- Excluir Evento (Botão)

- Excluir Recorrente (Botão)
